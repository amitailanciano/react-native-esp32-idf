# react-native-esp32-idf

provisioning wifi to esp devices

## Installation

```sh
npm install react-native-esp32-idf
```

## Usage

```js
import Esp32Idf from "react-native-esp32-idf";

// ...

const result = await Esp32Idf.multiply(3, 7);
```
